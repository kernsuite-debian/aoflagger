AOFlagger manual
================

This is the manual for the AOFlagger software suite.
The AOFlagger software can find and remove radio-frequency interference (RFI) in radio astronomical observations.
The Lua language is used for writing flexible flagging strategies. The tools are applicable to a wide range of telescopes.
